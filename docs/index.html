Investigation Workflow Timeline

1

Alert Triggered

Security Onion or Wazuh generates an alert from host or network telemetry.

2

Evidence Reviewed

Raw logs, source IP, usernames, event IDs, process data, and affected systems are validated.

3

MITRE Mapped

Activity is translated into ATT&CK techniques to standardize reporting and analysis.

4

Incident Summary Written

Severity, scope, evidence, and recommended next steps are documented as a SOC-style report.
