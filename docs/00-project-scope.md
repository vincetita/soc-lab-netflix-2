
# Project Scope

This project builds a complete SOC lab environment that simulates an enterprise network and demonstrates how attacks can be detected, investigated, and documented.

The lab includes:

- Active Directory environment
- Windows and Linux monitored systems
- Wazuh SIEM
- Security Onion network monitoring
- Kali attacker workstation
- Sysmon endpoint telemetry
- Docker-related deployment elements where relevant

The purpose of this lab is to demonstrate practical blue-team, SOC analyst, and detection engineering skills in a realistic environment.
