
# LinkedIn Showcase Post

I recently completed building an enterprise-style SOC lab environment to strengthen my blue-team and SOC analyst skills.

The lab includes:

- Windows Server 2025 Domain Controller
- Windows 11 domain workstation
- Ubuntu Linux server
- Wazuh SIEM
- Security Onion IDS
- Sysmon endpoint telemetry
- Kali Linux attacker simulation

In this project I implemented:

- Active Directory, DNS, and DHCP
- endpoint monitoring with Wazuh agents
- Windows telemetry using Sysmon
- network monitoring using Suricata and Zeek
- attack simulations including scanning, brute force, privilege escalation, persistence, and suspicious PowerShell activity
- MITRE ATT&CK detection mapping
- SOC investigation workflows
- incident reporting templates

This project helped me practice:

- log analysis
- SIEM correlation
- threat hunting
- detection engineering
- network intrusion detection
- Active Directory monitoring

What I liked most was combining endpoint visibility from Wazuh and Sysmon with network visibility from Security Onion to investigate attacks from multiple angles.

This project is documented step by step in my GitHub portfolio.
