
# Investigation Case — Windows Failed Logon

This case investigates failed Windows authentication attempts detected by the SOC monitoring system.

---

# Alert Source

Wazuh

Alert Type

Windows Failed Logon

Event ID

4625

---

# Attack Scenario

A simulated brute force attack was performed using incorrect credentials.

Command used


runas /user:corp\administrator cmd


Incorrect passwords were entered repeatedly.

---

# Investigation Steps

## Step 1 — Review Wazuh Alert

The alert identifies the host generating the failed logins.

Example details

User


administrator


Source IP


192.168.117.100


---

## Step 2 — Review Windows Security Logs

Event Viewer path


Windows Logs → Security


Event ID

4625

Example message


An account failed to log on


---

## Step 3 — Confirm Attack Pattern

Indicators

• repeated failed login attempts
• same source IP
• same user account

---

# MITRE ATT&CK Mapping

Technique

T1110

Credential Access

Brute Force

---

# Investigation Outcome

The failed login attempts were generated by the Win11Client system during the attack simulation.
